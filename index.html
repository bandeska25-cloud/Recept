Recepttár adatbázis

Recepttár alkalmazás - 7 technológia, 1 adatbázis

Ez a beadandó a receptes adathalmazt dolgozza fel: ételek, kategóriák, hozzávalók és mennyiségek jelennek meg egységes webes felületen.

A projekt célja

A Webprogramozás 1 kurzus előadás beadandójához készült alkalmazás ugyanazt a CRUD logikát több technológiai megközelítésben mutatja be. A téma a receptgyűjtemény, az alapadatokat az etel.txt, kategoria.txt, hozzavalo.txt és hasznalt.txt fájlok adják.

2 pont

CRUD tiszta JavaScripttel, memóriában tárolt recept tömbbel.

2 pont

React

Tech stack

Front-end: HTML5, CSS3, JavaScript, React. Back-end: PHP 8 + PDO, MySQL/MariaDB. HTTP: Fetch API és Axios. Deploy: Nethely.hu kompatibilis fájlszerkezet.
